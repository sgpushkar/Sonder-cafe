# ☕ Sonder Cafe (Demo Website)

A modern and aesthetic **cafe website demo** built using React and TypeScript.
This project focuses on UI design, layout structure, and smooth user experience for a cafe-style brand.

🔗 Live Demo: https://sonder-cafe.vercel.app/

---

## ✨ About the Project

Sonder Cafe is a front-end demo website designed to represent a stylish and minimal cafe presence online.
It highlights clean visuals, intuitive navigation, and a responsive layout suitable for real-world use cases.

---

## 🚀 Features

* 📱 Fully responsive design (mobile + desktop)
* ⚡ Fast performance powered by Vite
* 🎨 Modern and minimal UI design
* 🧭 Smooth navigation experience
* 📋 Menu-style layout for showcasing items
* 🔍 Basic SEO-friendly structure

---

## 🛠️ Tech Stack

* **React + TypeScript**
* **Vite**
* **CSS / Tailwind (if used)**
* **Vercel (Deployment)**
